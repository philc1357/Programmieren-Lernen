Man kann über HTML Text wie Quellccode aussehen lassen

Dazu verwendet man den Tag 

 <code></code>

Mann kann auch ohne CSS ein Element horizontal mittig plazieren

Dazu verwendet man den Tag 

 <center></center>

Beispiel: 

 <center>Das ist die Mitte</center>

geschrieben, sieht so aus:

Das ist die Mitte

Variablen Deklaration

let name = "wert"; - Der Wert kann verändert werden. Du kannst die Variable neu zuweisen.

const name = "wert"; - Der Wert kann nicht verändert werden. Es ist eine Konstante.
